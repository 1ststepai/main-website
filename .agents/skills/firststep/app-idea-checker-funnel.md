# 1stStep App Idea Checker Funnel

Use when building a funnel that helps prospects validate an app or automation idea.

Checklist:
- Ask for problem, audience, current workaround, desired workflow, data sources, budget/timeline.
- Return next-step fit: build, prototype, automate, defer.
- Capture lead details and permission to follow up.
- Make result useful before a sales call.

Do not:
- Pretend validation is definitive.
- Trigger paid AI/API calls automatically.

Output:
- Funnel flow, intake fields, result states, CTA, follow-up copy.
