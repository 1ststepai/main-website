# 1stStep Portfolio Case Studies

Use when turning 1stStep.ai projects into portfolio pages, case studies, or proof assets.

Checklist:
- Problem, constraints, solution, implementation, outcome, screenshots, stack.
- Separate real results from qualitative wins.
- Include client-safe language and permissions.
- CTA connects to similar project request.

Do not:
- Invent metrics or logos.
- Reveal private client data.

Output:
- Case-study structure, copy, visuals list, CTA, proof-risk notes.
